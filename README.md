#  Tic Tac Toe Game

This project is a simple Tic Tac Toe game written in Python.  
It includes both a "text-based console version" and a "GUI version" built using "Tkinter".


## Features

Two-player mode (Player X and Player O)  
Input validation for correct moves  
Detects win, draw, and invalid inputs  
GUI version using Tkinter 
Demonstrates Git branching and merging best practices  


## How to Play

## Console Version:
 Run the Python file:

   python tic-tac-toe.py

## GUI Version

After the console version, GUI version will start automatically,you have to run the file onle once.
